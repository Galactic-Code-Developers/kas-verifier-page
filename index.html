🔐 What This Verifies

  • Confirms authorship identity (primary author metadata match)
  • Validates document fingerprint (SHA-256)
  • Checks signature from KAS official key
  • Flags tampered or unauthorized documents

📄 File Naming

[AuthorLastName]_[ShortTitle]_v[Version]_SIGNED_KAS_[Date].pdf

Each signed document includes:

  • .pdf — signed paper
  • .sha256 — fingerprint
  • .sig — RSA signature
